API Documentation

This page is still under construction. See below for some basic information about our API to get started. If you are using our API at all or analyzing or visualizing Eyebrowse data, please let us know at eyebrowse@mit.edu. We'd love to hear about what you are up to and to showcase your work!

Eyebrowse History

The following returns all eyebrowse history events sorted by reverse chronological order. Bulletin messages posted on any page are included in the results. Pagination is included so you can go through the data more easily:
http://eyebrowse.csail.mit.edu/api/v1/history-data?format=json&offset=0&limit=20

You can find the eyebrowse data for a specific page:
http://eyebrowse.csail.mit.edu/api/v1/history-data?format=json&url=http://eyebrowse.csail.mit.edu/live_stream/?filter=following

or for a specific user:
http://eyebrowse.csail.mit.edu/api/v1/history-data?format=json&user__username=amyxzhang

You can also filter by start_time and end_time:
http://eyebrowse.csail.mit.edu/api/v1/history-data?format=json&offset=0&limit=20&start_time__gt=2014-07-23
